Do you and your partner fight over which movie to watch?

We can help! Our specially designed KNN model will take the title of a movie you like and recommend other movies that it thinks you will like.

The best part? Our model has been optimized to give you the best and most accurate results! We have even deployed our model on an easy to use website for you.

Content Based Filtering by Genre and Description

Content based filtering is a technique that uses feature similarities to make decisions. Content based filtering is often used for systems that recommend or make recommendations. In our case we used a content based filtering system to check for similarities between genres of movies to make recommendations. We also used the same approach to make recommendations based on similarities in movie descriptions. In the case of movie description based filtering term frequency was used. Term frequency answers the question of, how many times does this word appear in this document among the number of times all words appear in this document? In other words, how important is this word to this specific document?

Learn more about Content Based Filtering

Unsupervised Learning: KNN Model

In our second recommender model we used an unsupervised KNN approach to recommend movies based on the ratings they were given by users. In this instance we used the ratings feature and user information to predict a movie a user would like based on the rating that was given to a movie by other users. When a movie was given to this model it would compare the ratings of the movie to the clusters that it had created and find the nearest neighbour movie to the movie you searched. Other features besides ratings could have been used to create clusters.

Learn more about KNN

Supervised Learning: SVD

We also created a supervised learning model that predicts the rating of a movie. We used Singular Value Decomposition Matrix Factorization (SVD MF); using surprise python skicit. Singular Value Decomposition (SVD) is a widely used technique to decompose a matrix into several component matrices, exposing many of the useful and interesting properties of the original matrix. Matrix factorization is a class of collaborative filtering algorithms used in recommender systems. Matrix factorization algorithms work by decomposing the user-item interaction matrix into the product of two lower dimensionality rectangular matrices. The XGBoost Algorithm was used later on and fitted to this model. XGBoost is an implementation of gradient boosted decision trees designed for speed and performance.
